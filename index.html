About

Universitas Gadjah Mada menyelenggarakan berbagai program pendidikan meliputi program sarjana, pascasarjana, profesi, spesialis, dan diploma. Beberapa fakultas juga menyelenggarakan program internasional baik pada program sarjana maupun pascasarjana.

Program Sarjana

Program sarjana bisa diikuti oleh lulusan SMA/SMK/MA/MAK, mempunyai beban studi 144-160 sks dan dapat ditempuh dalam waktu 8-14 semester.

Program Sarjana Terapan

Program Sarjana Terapan menawarkan berbagai program studi yang berfokus dalam membekali mahasiswa dengan keterampilan yang relevan dengan tuntutan dunia kerja

Program Pascasarjana

UGM menawarkan program studi lanjutan di tingkat S2 dan S3 yang diselenggarakan di fakultas serta sekolah Pascasarjana.

Program Profesi

Program profesi diperuntukkan sebagai bagi lulusan program studi tertentu sebagai syarat untuk mendapatkan gelar keprofesian.
